# Goda Real Estate Group

Bienvenue dans le projet Goda Real Estate Group !

## Description

Ce projet est une page web pour Goda Real Estate Group, une agence immobilière proposant des services personnalisés au Maroc. La page web présente des informations sur les services offerts, les emplacements couverts, les horaires d'ouverture, ainsi qu'une galerie de photos et une barre de recherche.

## Installation

1. Clonez le référentiel sur votre machine locale : 

2. Ouvrez le fichier index.html dans votre navigateur web pour afficher la page.

## Utilisation

Naviguez à travers les différentes sections de la page pour découvrir les services offerts par Goda Real Estate Group. Utilisez la barre de recherche pour trouver des propriétés spécifiques. Explorez la galerie de photos pour voir des exemples de propriétés disponibles.
